Rekurzív algoritmus

A rekurzív algoritmus minden lépésben felezi a kitevőt, így a komplexitás \(\mathcal{O}(\log{n})\) lesz:

\begin{equation} a^n=\begin{cases} 1 &\ \text{ha $n=0$}.\\[.8em] \left(a^m\right)^2 &\ \text{ha } n > 0 \text{, } n \equiv 0 \text{ (mod 2)} \text{, ahol }m = \lfloor\frac{n}{2}\rfloor\\[.8em] \left(a^m\right)^2\cdot a &\ \text{ha } n > 0 \text{, } n \equiv 1 \text{ (mod 2)} \text{, ahol }m = \lfloor\frac{n}{2}\rfloor\\ \end{cases} \end{equation} 
int fpow(int b, int p) {
   if(p == 0) {
      return 1;
   }
   int res = fpow(b, p >> 1);
   if(p % 2) {
      return res * res;
   } else {
      return res * res * a;
   }
}

Iteratív algoritmus

Az iteratív algoritmus a kitevő bináris felírásán alapszik. Fenntartunk egy segédváltozót, ami az alap aktuális bitnek megfelelő hatványát tárolja, és ha ez a bit aktív, az eredményt megszorozzuk a segédváltozóval.
Például: \[3^{25}=3^{11001_{2}}=3^0\cdot 3^8\cdot 3^{16}=3^\left(2^0\right)\cdot3^\left(2^3\right)\cdot3^\left(2^4\right)\] 

int fpow(int b, int p) {
   int res = 1;
   int pow = b;
   for(; p; p >>= 1) {
      if(p & 1) {
         res *= pow;
      }
      pow *= pow;
   }
   return res;
}
